Finds the ladder (path) between two Wikipedia pages. 

This method implements the Breadth-\/\+First Search (BFS) algorithm to find the shortest path (ladder) between the start and target Wikipedia pages. It uses a queue to track the paths and an unordered set to track visited pages.
